# Démineur en Python 
Ce projet est une implémentation du jeu du démineur en Python avec la bibliothèque graphique **FLTK**.

## Présentation
Le jeu propose une interface graphique où le joueur doit :
- Découvrir toutes les cases sans tomber sur une mine 
- Utiliser des drapeaux pour marquer les cases suspectes
- Se fier aux nombres indiquant les mines adjacentes

## Fonctionnalités
- Interface graphique avec FLTK
- Plusieurs niveaux de difficulté
- Génération aléatoire des mines
- Système de drapeaux
- Détection de victoire / défaite
- Chronomètre intégré 
- Révélation automatique des zones vides

## Comment jouer
- Clique gauche sur une case pour la révéler
- Clique droit pour poser ou enlever un drapeau 
- Les nombres affichés indiquent combien de mines sont présentes autour de la case
- Si tu touches une mine, la partie est perdue
- Révèle toutes les cases sans mines pour gagner 
- Utilise les drapeaux pour marquer les cases que tu penses dangereuses

## Installation
- Télécharger tout les fichiers
- Placer les fichiers dans un même dossier et les images dans un sous-dossier
- Lancer le code dans un IED comme Thonny
